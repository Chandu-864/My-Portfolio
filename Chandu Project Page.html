This Weather App allows users to check the current weather of any city by fetching real-time data from the OpenWeather API. It displays the temperature, humidity, wind speed, and updates weather icons based on the conditions (e.g., clouds, rain, clear sky). Users can input a city name and get results by clicking the search button or pressing the "Enter" key. The app also handles invalid inputs, showing an error message for unrecognized cities. Built with HTML, CSS, and JavaScript, the app demonstrates my skills in API integration and responsive design.

I created a two-player Tic-Tac-Toe game that allows players to take turns marking cells with "O" or "X." The game board responds dynamically to user clicks, checking for win conditions after every move. If a player wins, an alert displays the winner, and the game is disabled. In the event of a tie, the game notifies the players and stops further interactions. A restart button is included to reset the game and allow players to start a new match. This project demonstrates my skills in DOM manipulation, event handling, and implementing game logic using JavaScript.
